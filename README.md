# Cocapn Marine ⚓

> **Marine sensor integration for CoCapn — NMEA 0183 parsing, PID autopilot, bathymetric recording, and deadband monitoring.**

> 📚 **Documentation:** [`PLUG_AND_PLAY.md`](./PLUG_AND_PLAY.md) · [`GETTING_STARTED.md`](./GETTING_STARTED.md) · [`ARCHITECTURE.md`](./ARCHITECTURE.md) · [`API_REFERENCE.md`](./API_REFERENCE.md) · [`LOW_LEVEL.md`](./LOW_LEVEL.md)

A Rust library for marine systems: parse NMEA sentences from GPS/depth/heading sensors, run a PID autopilot with anti-windup, record bathymetric data, and monitor heading deadbands.
